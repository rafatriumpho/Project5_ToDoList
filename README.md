\# Project 5 – To-Do List (Python)



This is my fifth Python project, built during Week 5 of my Python learning journey.



\## Overview



This program is a simple command-line to-do list application that allows the user to manage tasks with persistent storage.



\## Features



\- Add new tasks



\- Remove existing tasks



\- View all tasks



\- Save tasks to a file



\- Load tasks automatically when the program starts



\## Concepts Used



\- Lists



\- Loops (while, for)



\- Conditional statements (if/elif)



\- File handling (reading and writing)



\- String manipulation



\## How It Works



\- When the program starts, it reads tasks from Tasks.txt



\- The user selects an option from a menu:



&#x20; - 1: Add task



&#x20; - 2: Remove task



&#x20; - 3: View tasks



\- Any changes (add/remove) are immediately saved to the file
